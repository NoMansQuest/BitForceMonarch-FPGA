# Butterfly Labs BitForce Monarch FPGA Firmware

The FPGA firmware (Spartan 6 - XC75SLX-T) for Butterfly Labs BitForce Monarch Series. This repository contains:

- The design implemented in verilog source files
- The constraints UCF file

What it does not include:

- Xilinx ISE generated files P&R
- Flashing bitstream
- IPCores (BlockRAMs, AXI Bus and PCI-Express core)
